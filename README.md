# web-dev-final-project
## Front End
To set up the front end:
```
  cd find-your-meal
  npm install
```
To start the front end:
```
  cd find-your-meal
  npm start
```

## Back End
To set up the front end:
```
  cd server
  npm install
```
To start the front end:
```
  cd server
  nodemon server.js
```
